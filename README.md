# Math-4C-extra-credit
My extra credit project for Mission College's Math 4C (Linear Algebra) class.

I wrote a program that takes a user-inputted string and encodes and decodes it using an invertible matrix.

To run this program, clone this project to your local machine.

Run the following command: `python main.py`

The program will prompt you to enter a text string in all capital letters and no punctuation.
